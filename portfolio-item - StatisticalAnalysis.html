Proyecto Realizado durante el Bootcamp de Data Analytics en Ironhack

Generación de Predicciones con varios modelos de Machine Learning

Proyect del Bootcamp de Data Analytics en Ironhack

Durante este módulo se ve el tema de Estadística y Probabilidad.

Mi proyecto consiste en analizar un Data Set obtenido de Kaggle, el cual contiene las variables y precios de casas en venta en USA.

Se identifican las variables más importantes, las cuales afectan el precio de la casa y se generan varios modelos lineales y no lineales de Machine Learning para predecir el precio de las casas con diferentes variables.
