Proyecto de fin de modulo 2 del Bootcamp de Data Analytics en Ironhack

Para este proyecto utilizo el siguiente set de datos que contiene listados de películas y programas de televisión en Netflix, disponible en Kaggle

El objetivo de este proyecto es practicar la creación e interpretación de diferentes tipos de visualizaciones utilizando datos del mundo real.

Netflix es una aplicación que sigue creciendo más y más rápido con su popularidad, programas y contenido. Este es un análisis exploratorio de los datos junto con visualizaciones de los mejores insights que he encontrado.
